### About the Experiment 

This experiment helps learners understand and visualize the Bellman-Ford algorithm for finding shortest paths from a single source vertex in a weighted directed graph. The simulation supports both positive and negative edge weights, demonstrates edge relaxation over \(V-1\) iterations, and includes negative-cycle detection. Students can explore random graphs or build custom graphs manually, and learn through step-wise visualization and code walkthrough mode.
